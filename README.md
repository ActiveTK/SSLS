# About
SSLSは、パスワードの重複を許さない<b>最恐のセキュリティシステム</b>です。<br>
アカウント作成時におけるパスワード重複を適切に警告し、一人一人のユーザーが別のパスワードを使うことで、セキュリティの向上を図ることができます。
